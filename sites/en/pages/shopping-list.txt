=abstract start

Create a shopping list application.

See other <a href="/exercises">projects and exercises</a>.


=abstract end

<ul>
    <li>A one-line window to enter some text.</li>
    <li>An ADD button.</li>
    <li>A place where you can see the list of items added.</li>
</ul>

<ul>
    <li>Save the list in a file or a database.</li>
    <li>When opening the application again, show the list saved in the file.</li>
</ul>

<ul>
    <li>Allow the user to remove an item.</li>
    <li>Allow the user to edit an item.</li>
    <li>Allow the user to mark an item as "bought".</li>
</ul>

<ul>
    <li>Save the shooping list by date so it can be shown later.</li>
    <li>Clear the currently list to start over.</li>
    <li>Allow the user to see statistics on which items were bought.</li>
    <li>When adding new items, offer items that were previously bought.</li>
</ul>
