<a href="http://yaml.org/">YAML</a> that stands for <b>YAML Ain't Markup Language</b> is a human friendly data serialization format with implementations in many programming languages. (See <a href="https://en.wikipedia.org/wiki/YAML">Wikipedia on YAML</a>.)

It is similar to <a href="/json">JSON</a>, but looks much nicer and much more readable.

It is usually used for configuration files that need more power that an INI file.
