<h2>Install on Ubunutu</h2>

<code>
sudo apt-get -y install imagemagick
</code>


<code>
identify  image.png
file      image.png
</code>


<h2>Convert image</h2>

<code>
convert image.png -resize 300x400 new_image.png
</code>
